## 信息弹窗

早期JS是为服务于浏览器而生的，主要针对网页用来改进设计、验证表单、检测
浏览器、创建cookies等。

弹窗是浏览器中带的自有特性，JS可以根据开发者的需求与用户利用弹窗进行交
互操作。

### 弹窗种类

#### 提示弹窗

提示弹窗是最常用的，实现方式也非常简单，示例：

```
<script>
  alert('Hello World !');
</script>
```

#### 询问弹窗

询问弹窗有两个交互按钮，取消和确认。弹窗会返回用户的点击状态，true或者false。

```
<script>
  confirm('今天中午吃米饭');
</script>
```

#### 信息输入弹窗

propmt需要用户输入一些内容，我们可以通过该弹窗获取用户的输入信息。
   * 参数一：表示需要用户输入的提示信息
   * 参数二：默认输入内容（用户未输入内容时的默认信息）

```
<script>
  prompt('参数一', '参数二')
</script>
```
