## for遍历

此处用到数据类型[对象](./)，不了解此类数据格式的朋友请先戳[对象](./)进行学习。

### for in

常用于数组值遍历

1. 写法一

```
let arr = ['a', 'b', 'c', 'd'];

for (let i = 0; i < arr.length; i++){
    console.log( arr[i] )
}
```

2. 写法二 （简写）

```
var arr = ['a', 'b', 'c', 'd', 'e'];
var i = arr.length;
for(; i--;){
    console.log(arr[i])
}
```


### for of 

for of 和 for in用法类似，但是由于ES6中新增了Set和Map数据类型，传统遍历方式无法处理，此处可以用for of
