## 字符类型

字符类型表示用**单引号**或**双引号**包裹住的所有字符，字符类型用**String**表示。

### 字符定义

```
let v = 'Hello World !'
```

### 字符与字符串

字符通常表示单词或者单个文字，字符串表示多字符。

### 字符拼接

字符间可以进行拼接形成字符串，例如：

```
let a = 'Hello';
let b = 'world !';

a + b // hello world !
```

### 字符转换成数组

split

split(分割符号)

```
let v = 'hello';
console.log(v.split(':'))
```
