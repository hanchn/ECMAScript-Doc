## 条件过滤

### some
some用于过滤元素，是否有符合条件的元素

```
let arr = ['a', 'b', 'c', 'd', 'e'];
arr.some(v => v === 'a')
```


### every
some用于过滤元素，是所有条件都符合过滤条件
```
let arr = ['a', 'b', 'c', 'd', 'e'];
arr.every(v => v === 'a')
```


### filter
返回符合过滤条件的值
```
let arr = ['a', 'b', 'c', 'd', 'e'];
arr.filter(v => v === 'a')
```
