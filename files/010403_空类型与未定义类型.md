## 空类型与undefined

### 空类型

空类型用Null表示，空类型指空状态，暂时不赋值，先占个位（并非啥都没赋值）,此状态和undefined类似。

```
let v = null;
```

### 未定义类型

未定义类型表示啥都没赋值，区别空类型，也表示没赋值的状态。

```
let v; // v 的状态为undefined
```

### 特殊的null

null的状态很特殊，它既可以表示值类型也可以表示引用类型（[基础类型和对象](./)）,因为null表示暂未赋值。

### 布尔类型

布尔类型用Boolean表示，用于表示状态：true/false。

```
let v = true;
```
